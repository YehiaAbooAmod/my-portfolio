Group Policy & File Sharing Management

Project Overview

This project focuses on configuring shared drives and managing access through Group Policy Objects (GPOs). It demonstrates how to use GPOs to automatically map network drives, apply security restrictions, and deploy software from a shared source on a Windows Server environment.

Steps Completed

  • Created shared drives for Common and IT departments on the main server.
  • Set correct Share and NTFS permissions to restrict unauthorized access.
  • Configured two Group Policies — one for the IT OU and another for the Common OU.
  • Mapped each shared drive automatically using Group Policy preferences.
  • Created an additional policy named 7zip for software deployment from the shared folder.
  • Verified GPOs on client machines ensuring all mapped drives and deployments work correctly.
